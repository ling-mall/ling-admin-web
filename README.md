# ling-admin-web

ling-mall 项目后台前端

## 安装

克隆仓库

```bash
git clone https://github.com/ling-mall/ling-admin-web.git
```

安装依赖

```bash
pnpm i
```

启动

```bash
pnpm run dev
```
